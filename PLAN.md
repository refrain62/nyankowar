# Web版「ねこねこ大戦争」風ゲーム 開発最終状況

## 1. 技術スタック
- **フレームワーク:** React 19 (TypeScript)
- **リンター/フォーマッタ:** Biome (高速な静的解析と整形)
- **設計パターン:** Controller-System-View (疎結合アーキテクチャ)
- **描画:** Pure HTML5 Canvas
- **音声:** Web Audio API (Stable Interface & ポーズ対応)

## 2. 実装済み機能ハイライト

### アーキテクチャ・最適化 (最新) 🚀
- [x] **Controller の抽出 (`useGameController`):** 
    - ユニット召喚、レベルアップ、状態遷移などの「操作ロジック」を JSX から完全に分離。
- [x] **ロジックのサブモジュール化 (`gameSystems.ts`):**
    - 経済、戦闘、スポーン、クールダウンといった「計算ロジック」を独立モジュール化。
    - 巨大な関数を排除し、各システムの責務を明確化。
- [x] **純粋な View (`Game.tsx`):**
    - コンポーネントはレイアウトとイベントの委譲のみを担当。

### システム & UI ⚙️
- [x] **ポーズメニュー (Pause Menu):** 
    - 「RESUME」「RESTART」「QUIT」の3オプションを搭載。
- [x] **useEffect の徹底排除:** 
    - リソースの破棄以外に useEffect を使わない「イベント駆動型」を維持。

### ビジュアル & キャラクター演出 🎨
- [x] **全ネコ・手足完全実装:** 全ユニットに動く手足を実装。
- [x] **数式によるエフェクト:** 火炎放射、閃光、爆発などをコードのみで表現。

## 3. 総括
「見た目（View）」「操作（Controller）」「演算（System）」が三位一体となりつつも、それぞれが独立して動作する疎結合な設計を確立しました。
これにより、将来的な新ユニット追加や特殊スキルの実装、さらにはマルチプラットフォーム展開も容易な、スケーラブルなコードベースとなっています。
