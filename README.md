# ねこねこ大戦争プロトタイプ

「にゃんこ大戦争」のプレイ感を React + TypeScript + Canvas で再現した本格タワーディフェンス・プロトタイプです。

## 🌍 ステージ紹介
全4章の構成。ステージごとに色彩だけでなく、リズムや音色の異なる専用BGMが流れます。

## 🐱 味方ユニット (全8種・手足完備)
にゃんこ、タンク、バトル、キモねこ、ウシねこ、ネコノトリ、フィッシュ、トカゲ。
全てのユニットに動く手足が実装され、戦場を生き生きと（そしてシュールに）進軍します。

## 👹 敵ユニット (全4種)
| ユニット名 | 特徴 |
| :--- | :--- |
| **わんこ** | 基本の敵。数で攻めてくる。 |
| **へび** | 高速移動。波打つ動きで接近し、舌をチロチロ出す。 |
| **カバちゃん** | 高HPの壁役。大きな口で範囲噛みつき攻撃。 |
| **くま** | 最凶のボス。巨大な腕のなぎ払いで前線を破壊。 |

## ✨ 技術的特徴
- **ピュアコード・グラフィック:** 画像・音源ファイルを一切不使用。全てコード計算で描画・合成。
- **没入型UIシステム:** バトル開始時に操作パネルがフェードインする動的なUI。
- **高度な物理・演出:** 充填音を伴うにゃんこ砲や、火炎放射パーティクル等のリッチな演出。

## 🎮 遊び方
1. **タイトル:** 黄金の「CLICK TO START」をクリック。
2. **ステージ選択:** コンパクトなパネルから章を選んで出撃。
3. **バトル:** 8種のユニットを駆使して敵の城を落とせ！
4. **戻る:** 画面右上のボタンから、いつでもメニューへ復帰可能。
